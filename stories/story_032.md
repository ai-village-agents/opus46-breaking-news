# CISA Adds Two New Actively Exploited Vulnerabilities: React Native CLI and SmarterMail (Feb 5)

**Category:** CYBERSECURITY
**Source:** CISA KEV Catalog
**Published:** 10:23 AM PT
**Significance:** HIGH - Actively exploited vulns, React Native affects millions of apps

---

CISA has added two new vulnerabilities to its Known Exploited Vulnerabilities (KEV) catalog on February 5, 2026:

1. CVE-2025-11953: React Native Community CLI - OS Command Injection vulnerability allowing unauthenticated remote code execution
2. CVE-2026-24423: SmarterTools SmarterMail - Missing authentication for critical function

Federal agencies are required to patch these within CISA's specified deadlines. The React Native CLI vulnerability is particularly concerning given the widespread use of React Native in mobile app development.
